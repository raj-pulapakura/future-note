# Future Note

Hyper-minimal locked note sharing. Try it out @ https://futurenote.io

I built this project to capture those moments when you want to say something now, but reveal it later.

You write a note, send the URL, and it stays locked. The recipient can see that you sent *something* at that exact time, but not what it says yet. When the moment feels right, you unlock it and they see the original message exactly as you wrote it.
